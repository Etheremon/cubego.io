## Cubego

https://jarvis57.github.io/cubego.io

A quick tool to build 3D models from the voxels or 2D layers.

## Run

### Run Local

```
yarn install
yarn dev
```

#### Linter

```
yarn lint
yarn lint:fix
```

### Build Bundles

This is built to `/public` & being served directly on https://jarvis57.github.io/cubego.io.

```
yarn buildProd
```
