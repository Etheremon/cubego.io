## Run

### Run Local

```
yarn install
yarn dev
```

#### Linter

```
yarn lint
yarn lint:fix
```

### Build Bundles

This is built to `/public` & being served directly on https://jarvis57.github.io/cubego.io.

```
yarn buildProd
```
